Day 35 project for Paul Hudson's 100 Days of SwiftUI course, but took two full days working overtime because of the ideas I wanted to integrate. Learned about .sheet, published variables, speaking across different views/structs, cleanly creating dynamic rows/columns of objects to appear on the screen without hardcoding every single position (oops), timers and using the concepts learned in the last few units. I also learned how to work a bit more with arrays to tie behavior to their changes in contents.
